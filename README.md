# 📦 Inventory Management System (Java Swing)

A desktop-based **Inventory Management System** built using Java and Swing, designed to manage product stock efficiently with a clean and interactive user interface.

## 🚀 Features

* 🔐 Login Authentication
* 📦 Add / Update / Delete Products
* 📊 Real-time Stock Monitoring
* 🔴 Low Stock Alerts (highlighted in table)
* 🔍 Search & Filter Products
* 📁 Export Inventory to File
* 🧾 Simple Billing System
* 📂 Category-Based Inventory
* 📈 Dashboard Summary (Total Items, Quantity, Low Stock)

---

## 🖥️ Tech Stack

* **Language:** Java
* **GUI:** Swing (Java AWT & Swing)
* **IDE:** IntelliJ IDEA / Eclipse
* **Data Storage:** In-memory (HashMap)
* **File Handling:** Java FileWriter

---
